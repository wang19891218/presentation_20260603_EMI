Times Square–calibrated benchmark — scaled to 9,000.

Bilinear stiffness

9,000-building feature grid

Open with the map: 85 real OSM buildings near Times Square (50–123 m) — these are the REFERENCE whose geometry and height distribution calibrate the synthetic benchmark. The 160 benchmark cases are SYNTHETIC: 80 parametric building configs (16 heights × 5 damping ratios) × 2 wind realizations each, split 128 train / 16 val / 16 test by case. Five conditioning features: height H, first-mode frequency f, post-yield ratio γ, damping ratio ζ, density ρ_B. Structural model (figure_007, center): bilinear storey stiffness — linear up to yield displacement δ_y, then post-yield stiffness = γ k_0, γ ∈ [0.05, 0.10] per Hong 2004. Buildings remain linear under 300-year MRI wind. Feature space (figure_008, right): expanded 9,000-building full-factorial grid over 7 features — height (8 levels: 50–120 m), width (5: 15–35 m), depth (5: 15–35 m), damping ratio (5: 1–5%), density (3: 160–240 kg/m³), floor count (3: 10/15/20 stories), fundamental frequency derived by Rayleigh. Split 7,200 training / 900 validation / 900 test. The accent column header ("9,000-building feature grid") draws the eye to this key scaling result. Wind loading: Hilbert-wavelet simulation (Wang 2020) → per-floor forces via quasi-steady theory, drag coefficients from the Tokyo Polytechnic University aerodynamic database. Record duration: 60 s.
