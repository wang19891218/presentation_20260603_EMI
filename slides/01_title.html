HyperNetwork with Transformer-Based Weight Generation for Regional Wind-Induced Structural Response Estimation

Haifeng Wang — Washington State University

Welcome. I'm Haifeng Wang from Washington State University. This talk is about a single trained model that generates building-specific response predictors — no per-building retraining. Core idea: we read a compact structural feature vector and emit the weights of a response-predicting LSTM in one forward pass.
